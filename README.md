# <font style="color:#1aad19;">02 系统功能需求分析</font>
## 员工功能
(1)采购物资计划：员工可以发起采购计划，填写采购明细，发起采购，等待管理员审核，审核通过后，进行入库，生成入库单。在采购审核通过前，用户可以进行取消采购计划。

(2)提交申领信息：基于实际工作需求，员工可在系统中填写物资出库申请，等待管理员审核，审核通过后，会生成出库单。在申请确认前，可以进行取消申领。

(3)物资入库管理：员工发起的采购计划，生成入库单后，可将商品入库到具体的库位位置，进行入库完成。入库完成后，库存会增加。

(4)物资出库管理：员工发起的申领信息，管理员审核通过后，会生成出库单，出库确认后，库存会相应减少。

(5)库存盘点：员工可根据工作安排，对物资进行盘点。在盘点过程中，核对实际库存数量与系统记录数量是否一致，若存在差异，会生成对应的库存变动。

(6)库存查询：员工可查看公司的物资信息，可通过库房、商品条件筛选查看。

(7)库存变动记录：当商品进行入库、出库和盘点时，都会生成相应的库存变动记录。

---

## 管理员功能
(1)可视化分析：系统根据环形图展示库位使用情况，横向柱状图展示单据完成情况、库存预警商品。饼状图展示商品品牌分布，柱状图展示商品价格区间分布等。

(2)账号管理：管理账号信息，可新增账号，修改账号，导出账号信息。

(3)员工管理：管理员工信息，对员工信息进行增删改查。

(4)供应商管理：管理员可以维护供应商信息，所有的货源都由管理员进行把控。

(5)部门管理：对部门信息进行增删改查操作。

(6)库存查询：管理员可查看公司的物资库存情况。

(7)库存变动：当物资进行变动时，都有一笔对应的库存变动记录、

(8)采购计划管理：管理员收到员工提交的采购申请后，需在系统中进行审批操作。审批时，可查看采购申请的详细信息。若批准，系统将自动生成入库单；若拒绝，则需注明拒绝原因，反馈给员工。

(9)物资出库管理：对于员工提交的物资领用申请，管理员需进行审批。依据企业的物资管理制度和库存情况，判断是否批准领用。若批准，系统会自动生成出库单；若拒绝，需告知领用人拒绝原因。

(10)物资类型管理：管理员有权在系统中进行物资类型的添加、修改和删除操作。可根据企业的实际需求，定义物资类型的名称、描述、所属分类等属性，对物资进行科学分类，以便于物资的管理和查询。

(11)商品管理：管理员能够对商品物资的基本信息进行全面管理，包括添加新物资、修改物资的名称、品牌、价格等信息，以及删除不再使用的废弃物资。

(12)库房管理：管理员可管理库房信息。

(13)库位管理：管理员可管理库位信息。
